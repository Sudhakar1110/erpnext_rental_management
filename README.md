# Vehicle Rental Management (ERPNext v15+)

Complete fleet & rental management built on Frappe/ERPNext v15+.
Feature-parity with **Rentsyst** (rentsyst.com) and **Coastr** (coastr.com).

## Quick Install
```bash
bench get-app https://github.com/<your-org>/vehicle_rental_management.git
bench --site <site> install-app vehicle_rental_management
bench --site <site> migrate
bench restart
```

See [SETUP.md](./SETUP.md) for full installation guide and feature mapping.
